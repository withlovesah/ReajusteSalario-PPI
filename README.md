# Atividade 1 - Reajuste de Salário
**Análise e Desenvolvimento de Sistemas - Unoeste**

## 🎯 Objetivo
Cálculo automático de reajuste salarial para 3.000 funcionários de empresa de Presidente Prudente-SP.

## 📊 Funcionalidades
- Validação de dados via query string na URL
- Cálculo dinâmico baseado em regras de idade, sexo e tempo de empresa
- Interface responsiva e visual

## 🛠️ Stack
Node.js | Express.js | HTML/CSS | Vercel

## 🚀 Como usar
Copie e cole na barra do navegador, mudando os números pros seus dados:
http://localhost:3000/?idade=25&sexo=F&salario_base=1700&anoContratacao=2014&matricula=12345

## 📋 Requisitos Validados
✅ Idade > 16  
✅ Salário válido  
✅ Ano > 1960  
✅ Matrícula > 0
